HOW TO BUILD FOR ARM64:

Visual Studio:
  1. Open HomeKitADK.sln
  2. Platform dropdown → Select "ARM64"
  3. Configuration → "Release"
  4. Build → Rebuild Solution

CMake:
  cmake --preset windows-arm64-release
  cmake --build --preset windows-arm64-release

Dependencies for ARM64:
  vcpkg install openssl:arm64-windows
  .\Scripts\Build-Dependencies.ps1 -Architecture ARM64
